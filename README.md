# 🧠 Pokemon Manager

Sistema full stack para gerenciamento de Pokémons.

## 🚀 Stack

- Frontend: React + TypeScript + Tailwind
- Backend: MuleSoft (APIKit)
- Banco: PostgreSQL (Docker)

## 🧩 Arquitetura

Frontend → Experience API → PokeAPI + Database

## 📌 Objetivo

Permitir buscar, visualizar e salvar Pokémons.
